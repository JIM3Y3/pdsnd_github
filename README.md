### Project Title
Bikeshare Data Analysis Program

### Description
This project explores bikeshare data from three major US cities using Python.
It allows users to filter data by city, month, and day of the week, and then diaplys statistics about:
- Most common travel times
- Most popular stations
- Trip duration statistics
- User information

The program demonstrates the use of Python for data analysis and Git/Github for version control.
